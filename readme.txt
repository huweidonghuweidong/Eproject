网站为文具一号电商网站，在网站中做了以下功能：
banner图自动轮播效果，八个小的简介图轮播效果，二级菜单，根据点击的信息，切换不同的数据，通过监听鼠标滚轮事件实现滚动到一定的距离时，出现楼层效果，和上面的搜索输入框，在首页的各楼层中，每个楼层右边的六个图片都是通过json数据加载进来，点击之后就会进入到商品列表页面
在每个页面中都可以点击左上角的登录注册按键，进入登录注册页面，当登录成功后，就会自动返回首页，然后在各个页面的左上角就会显示你的用户名和注销按键，点击注销可以退出登录
在商品详情页面中的六十个数据都是json数据通过用jQuery的ajax加载进来，当点击不同的商品时，会跳到同一个商品详情页面
在商品详情页中，商品的图片，名字，价格都是根据在商品列表中点击的商品数据加载的，点击加入购物车后就会有弹框提示加入成功，然后点击右上角的“我的购物车”进入到购物车页面，可以查看自己加入购物车的商品
在购物车页面中，可以勾选想要购买的商品，下面的框中会实时显示总金额，商品的那一行还有添加数量，和删除功能
